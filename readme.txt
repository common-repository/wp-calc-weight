WP-Calc-Weight: calculator tool for Ideal weight calculation.

== Description ==

This online calculator allows calculating the ideal weight of the body, based on several hypotheses, which take into account the physiological parameters of a person.

= Developer's plugin page =
[Jarvis365](https://www.jarvis365.com/)

== Installation ==

1. Install WP-Calc-Weight by uploading the files to your server
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the WP-Calc-Weight settings and set your options.
4. Go to any page and place WP-Calc-Weight shortcode.
